Оборудование для сушки проб
Электрошкаф для сушки проб ЭШС-10.8.12.
Электрошкаф для сушки проб с рециркуляцией воздуха предназначен для сушки влажного сырья (проб, промпродуктов, шламов и т. п.) при окислительной атмосфере при температуре 110 Со. Пробы сушатся в стандартных оцинкованных или нержавеющих противнях с размерами 350*475*80 мм. Противни заказываются отдельно.
Электрошкаф для сушки проб. ЭШС-20.8.12.
Электрошкаф с рециркуляцией воздуха предназначен для сушки влажного сырья (проб, промпродуктов, шламов и т. п.) при окислительной атмосфере при температуре 110 Со. Предусмотрена возможность работы как одновременно всех секций, так и одной. Пробы сушатся в стандартных оцинкованных или нержавеющих противнях с размерами 350*475*80 мм. Противни заказываются отдельно.
Электрошкаф для сушки проб. ЭШС-30.8.12.
Трех секционный сборный сушильный шкаф с рециркуляцией воздуха предназначен для сушки влажных геологических проб при окислительной атмосфере при температуре до110 Со. Предусмотрена возможность работы как одновременно всех секций, так двух, одной. Пробы сушатся в стандартных оцинкованных или нержавеющих противнях с размерами 350*475*80 мм. Противни заказываются отдельно.
Противень оцинкованный
Предназначен для размещения проб в сушильных шкафах. Материал противня – Ст. ОЦ 0,6мм
Противень нержавеющий
Предназначен для размещения проб в сушильных шкафах. Материал противня – Сталь AISI409 0,8мм.
